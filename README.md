# GY-PROJECT-1-PRICE
QUESTION -> What if an ai tries predict a human’s *next reasoning step* before the human mind executes it?
This project does not aim to predict outcomes like usual.
Instead, it models the sequence of mental actions people take while solving problems. 
Most AI systems focus on predicting results.
Human reasoning follows a set of intermediate steps: recalling rules, testing examples, hesitating, switching strategies. These transitions are rarely modeled directly.
This project tends to explores whether such transitions:
- are predictable at all
- exhibit statistical regularities
- or break down under uncertainty.
